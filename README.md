# 📊 Sales Data Analysis Project

## 🔹 Overview

This project focuses on analyzing sales data using Excel and Power BI to generate meaningful insights and support business decisions.

## 🔹 Tools Used

* Microsoft Excel (Data Cleaning, Pivot Tables, Charts)
* Power BI (Dashboard & Data Visualization)

## 🔹 Key Insights

* Identified monthly sales trends
* Found top 10 best-selling products
* Analyzed region-wise sales performance

## 🔹 Dashboard Features

* Interactive Power BI dashboard
* Filters (Slicers) for Region and Category
* Clean and professional layout

## 🔹 Files Included

* Excel Analysis File (.xlsx)
* Power BI Dashboard (.pbix)
* Dashboard Screenshot
